# CollabNav

## How to clone the repository
    git clone https://github.com/SynergicRobotics/CollabNav.git

## How to compile
    mkdir build
    cd build
    cmake ..
    make

### For debug
    mkdir build
    cd build
    cmake -DCMAKE_BUILD_TYPE=Debug ..
    make

## Code style
We are going to use [ROS C++ Code Style](http://www.ros.org/wiki/CppStyleGuide).
